# Функция «График проверок»

Формирует **график проверок качества работы соцработников** на полугодие в формате
`.xlsx` (через `openpyxl`, без Office). Архитектура — MVC
([АРХИТЕКТУРА.md](../АРХИТЕКТУРА.md)).

- **Пакет:** `app/features/grafiki/`
- **Ключ функции:** `grafiki`
- **Вход:** соцработники из общей базы (по отделению) + отметки «самоконтроль» в окне.
- **Выход:** `.xlsx` — таблица «недели полугодия × соцработники» с отметками проверок.

---

## Что генерируется (по образцу `графики/Графики отд№9/`)

- Шапка «Утверждаю» (директор филиала), заголовок «График проверок качества работы
  социальных работников отделения № N на {1/2} полугодие {год} года», подпись зав. отделением.
- **Столбцы** — недели полугодия (по четвергам: июль 3,10,17,24,31; … дек 4,11,18,25 —
  всего ~26), сгруппированы под названиями месяцев.
- **Строки** — соцработники (№ + ФИО).
- В ячейках «**///**» — запланированная проверка на эту неделю; у соцработников на
  самоконтроле — «**самоконтроль**» (без проверок).

---

## Правило ротации

Активные (не на самоконтроле) соцработники идут по списку; **каждую неделю отмечается
следующий по кругу** (`service.build_schedule`): `marks[(индекс_работника, неделя)]='///'`,
работник = `active[неделя % len(active)]`. До конца полугодия круг повторяется (≈2–3 раза).
Соцработники на самоконтроле исключаются из ротации и помечаются «самоконтроль».

---

## Слои

- **`service.py`** (Controller) — `half_year_weeks(half, year)` (недели по четвергам),
  `build_schedule(workers, half, year)` (ротация), `generate(out, workers, half, year, …)`.
  `DEFAULTS` — организация/директор/зав./№ отделения (правятся в окне).
- **`writer.py`** (Model/IO) — `openpyxl` строит `.xlsx` с нуля: шапка, заголовок,
  объединённые заголовки месяцев + строка дат, строки соцработников, границы, подпись,
  альбомная печать «в 1 страницу по ширине».
- **`storage.py`** — отделения и соцработники из общей базы (`list_departments`, `soc_workers`).
- **`gui.py`** (View) — выбор отделения, полугодие (1/2) + год, поля №/директор/зав.,
  список соцработников с галочками «самоконтроль», «Сформировать».

---

## Сохранение

Сформированный файл, как и у всех функций, **сохраняется копией в базу** (архив
«Сохранённые документы», `core/documents.py`) — можно открыть/пересохранить позже.

## Проверка

Отдельной self-test команды нет (вход — из базы). Проверено: 2-е полугодие 2025 → 26 недель,
26 отметок по ротации, «самоконтроль» проставляется; файл открывается в Excel.
