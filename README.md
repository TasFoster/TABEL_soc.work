# TABEL — генератор офисных документов для отдела соцобслуживания

Настольное Windows-приложение (Python + Tkinter), которое формирует заполненные
офисные таблицы и документы для отдела социального обслуживания. Одно окно-лаунчер
(«главное меню») открывает независимые функции, каждая из которых выдаёт готовый
файл Excel / ODS / ODT — **без установленного Microsoft Office** (все писатели
файлов на чистом Python: `xlwt`, `odfpy`, `openpyxl`).

## Функции

| Функция | Что делает | Формат |
|---|---|---|
| **Табель** | Т-13, табель учёта рабочего времени | `.xls` |
| **Реестр** | Реестр выплат из входных `.xls` | `.ods` |
| **Приложение к табелю** | Нагрузка по дням на соцработника, перераспределение при отсутствии | `.xls` |
| **Проезд** | Пакет на компенсацию проезда (маршрутный лист + заявление), OCR билетов | `.ods` |
| **Услуги-Деньги** | Отчёт «услуги и деньги» из отчёта «071» + бесплатники/частичники | `.xlsx` |
| **График проверок** | Полугодовой график проверок соцработников (ротация) | `.xlsx` |
| **Протокол** | Протокол методчаса отдела | `.odt` |
| **Отчёт по госзаданию** | Отчёт о выполнении государственного задания по соцработникам | `.ods` |

Каждый созданный файл дополнительно архивируется в БД (окно «Сохранённые документы»).

## Структура

- **`tabel_app/`** — исходный код приложения (MVC / SOLID). Запуск: `python tabel_app/main.py`.
- **`tabel_app/docs/`** — справочная документация (архитектура, схема БД, по одному файлу на функцию).
- **`neuro/`** — отдельный эксперимент (PyTorch CRNN OCR), **не подключён** к приложению.
- Корневые `*.xls` / `*.xlsx` и папки `Реестрыы/`, `Проезд/`, `услуги деньги/`,
  `графики/`, `протокол/`, `Отчет соц раб/` — образцы входных и эталонных файлов.

## Запуск и сборка

```powershell
cd tabel_app
pip install -r requirements.txt
python main.py                                                    # запуск (главное меню)
powershell -ExecutionPolicy Bypass -File build.ps1               # сборка полной Tabel.exe
powershell -ExecutionPolicy Bypass -File build.ps1 -Variant lite # сборка облегчённой версии
```

Подробности — в [`tabel_app/README.md`](tabel_app/README.md),
[`CLAUDE.md`](CLAUDE.md) и [`tabel_app/docs/`](tabel_app/docs/).

## Зависимости

`xlrd`, `xlwt`, `odfpy`, `openpyxl` (см. `tabel_app/requirements.txt`).
Microsoft Excel / офисный пакет **не требуется**.
